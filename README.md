RSP-Discord-Bot
=================

Playing Rock Scissors Paper Game with Discord Bot by  Python
-------------------------------------------------------------

# 기능
- 다인 가위바위보 지원 디스코드 봇
  - 참여인원 집계
  - 게임 실시
  - 등수 및 결과 출력
- 2인 묵찌빠 지원 디스코드 봇
  - 상대 입력
  - 게임 승락
  - 게임 실시
  - 결과 출력
 
# 명령어
- !게임시작
   - !기본게임
     - !가위 / !찌
     - !바위 / !묵
     - !보 / !빠
   - !묵찌빠 (닉네임)
     - !가위 / !찌
     - !바위 / !묵
     - !보 / !빠
